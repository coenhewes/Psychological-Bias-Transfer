# Sources & Provenance

## Data
- **Treatment/control corpora** (`data/processed/synthetic_treatment.jsonl`,
  `synthetic_control.jsonl`): synthetic first-person posts generated by Gemini
  (`data/synthetic_generator.py`) from a fixed list of 24 everyday stressors, 10 posts
  per (marker × batch). ~15.2k tokens each.
  - Marker operationalization in the generator prompt matches
    `evaluation/marker_definitions.py` exemplars.
  - Control matching via subreddit pairs (anxiety/ptsd/domesticviolence →
    relationships/assistance), recorded in `data/processed/build_manifest.json`.
- **Raw reddit-matched sources** (`data/processed/treatment_corpus.jsonl`,
  `control_corpus.jsonl`): the pre-synthetic matched corpus used to seed/validate the
  synthetic generation. Kept for provenance.
- **Provenance gap:** the synthetic generator used a hardcoded API key (now stale) in
  `data/synthetic_generator.py`. The generated corpora are committed and reproducible
  from those files; regeneration would need a current Gemini key. No secret is live.
